# SHIFT // ZERO

> Gravity is your only weapon.

SHIFT // ZERO is a premium one-finger 2D action puzzle game focused on fast reflexes, creative mechanics, and polished gameplay.

Every few moments, the rules change.

Flip gravity.
Avoid deadly obstacles.
Master evolving mechanics.
Reach impossible scores.

---

## Features

- One-finger controls
- Dynamic gravity mechanics
- Procedurally evolving challenges
- Beautiful neon visuals
- Smooth 60 FPS gameplay
- Responsive for phones, tablets, and desktop
- Offline support
- Achievements and progression
- Unlockable skins and visual effects
- Regular content updates

---

- Desktop (Web)
